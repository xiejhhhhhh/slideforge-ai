# Prompt: Revise HTML Slide Layout

Please revise the HTML slide deck based on visual inspection.

Important manual correction rules:

1. Paragraph line spacing should be around 1.25. Do not use cramped spacing like 0.8 or 1.0 for dense Chinese text.
2. Avoid bad line wrapping. If one or two characters are left alone on a new line, adjust the text box, font size, or wording.
3. Same-level content must use unified font size, color, border, padding, and spacing.
4. Use the slide area fully. Avoid excessive bottom whitespace.
5. Keep all blocks aligned.
6. Check every slide in 16:9 display.
7. Tables must be fully visible.
8. Images must not be distorted.

Output the corrected HTML only.
