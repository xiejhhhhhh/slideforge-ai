# Prompt: Convert Markdown Outline To HTML Slides

Convert the Markdown PPT outline into a 16:9 HTML slide deck.

Requirements:

1. Use one `.slide` section per PPT page.
2. Keep the canvas fixed at 16:9.
3. Use separate text boxes for separate bullet points.
4. Use real figures if image paths are provided.
5. Add slide numbers and footer labels.
6. Avoid text overflow.
7. Use paragraph line height around 1.25.
8. Keep same-level content visually consistent.
9. Make the slide feel full but aligned.
10. Put speaker notes in a structured comment or notes block.
