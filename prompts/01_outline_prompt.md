# Prompt: Build A PPT Markdown Outline

You are helping me create a presentation.

Input materials:

- topic:
- audience:
- presentation time:
- slide count:
- required style:
- source notes:

Please create a Markdown PPT outline.

Requirements:

1. Use one section per slide.
2. Each slide must include:
   - slide title
   - core message
   - 3 to 5 content blocks
   - suggested figure or visual
   - speaker notes
3. Keep the logical flow clear.
4. Avoid long paragraphs on slides.
5. Put dense explanation in speaker notes.
