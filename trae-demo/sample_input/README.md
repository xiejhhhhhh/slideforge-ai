# 示例素材说明

本目录提供 Demo 体验用的示例素材。

## sample_requirement.txt
示例 PPT 需求文本。在 Demo 第 1 步点击「载入示例需求」按钮可自动填入，或手动复制此文件内容到需求输入框。

## 示例图片
建议准备 1-3 张与本主题相关的图片（如流程图、截图、示意图），在第 1 步上传。Demo 会用 AI 视觉模型分析图片内容，建议放在哪一页。

支持的图片格式：PNG / JPG / JPEG / GIF / WebP。

## 示例 PPT 模板
建议准备一个 .pptx 模板文件（任意学术或商务模板均可），在第 1 步上传。Demo 会解析模板的：
- 比例（16:9 / 4:3）
- 主色调
- 字体
- 标题/正文字号
- 页脚
- 适合的页面类型

后续生成的 HTML 幻灯片和导出的 PPTX 会尽量套用模板风格。

如果没有模板，Demo 会使用默认的学术简洁风格（紫色系，16:9）。
