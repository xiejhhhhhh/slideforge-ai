第 0 步：配置 AI 连接

首次使用需要填写你的 AI API 信息。数据只存在你的浏览器里，不会上传到服务器。

什么是 API Key？ 它是调用 AI 的凭证。
支持 OpenAI、DeepSeek、智谱 GLM、Moonshot 等兼容接口。
例如 DeepSeek：访问 platform.deepseek.com 注册后在 API Keys 页面创建。
小米 MiMo：Base URL 填 https://token-plan-cn.xiaomimimo.com/v1，普通测试建议先用 mimo-v2.5；mimo-v2.5-pro 是推理模型，短测试容易只返回 reasoning 内容。

第 1 步：输入 PPT 需求

描述你要做的 PPT，可上传参考图片和 PPT 模板（可选）。

🖼️
点击或拖拽图片到这里（支持多张）
📊
点击上传 .pptx 模板文件

第 2 步：Markdown 大纲

AI 根据你的需求生成 PPT 大纲，你可以让 AI 修改或重新生成。

📋点击下方按钮生成大纲
💬 让 AI 修改大纲
AI 会基于当前大纲和你的指令修改，保留你满意的部分。

第 3 步：HTML 幻灯片预览

AI 根据大纲生成 16:9 HTML 幻灯片，套用模板风格，并已结合图片布局建议插入用户图片。可逐页预览、对话修改。

第 0 / 0 页
🖼️点击下方按钮生成 HTML 幻灯片
💬 让 AI 修改幻灯片
AI 会重新生成所有页的 HTML，保留你满意的修改方向。

第 4 步：演讲稿备注

基于已确认的 HTML 幻灯片和大纲，AI 逐页生成演讲稿（会填入 PPTX 备注栏）。可逐页查看、直接编辑、或让 AI 整体调整。

�点击下方按钮生成演讲稿
💬 让 AI 调整演讲稿
AI 会基于当前演讲稿和你的指令调整，保持页数与大纲一致。

第 5 步：导出 PPTX

生成真实 .pptx 文件下载。导出方式根据是否上传模板自动选择。

💬 让 AI 调整导出内容（备注/标题/要点精简）
AI 会调整大纲数据（标题/要点/备注），然后你可以重新导出 PPTX。
